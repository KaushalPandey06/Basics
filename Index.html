printf();

In C-programming printf() is used to print any statement. In the simple braces[( )] followed by printf keyword we write our statement within double quotes[" "]. After writing our statement we must terminate the sentence by using semicolon[ ; ] after closing the simple braces.

printf(" statement to be print ");

#include< stdio.h >
void main()
{

printf(" ");

}

output

The semicolon[ ; ] is a special character used in C-language to terminate the following statement, it is also known as statement-terminator.

apart from printing only statememts we can print value of a varible by using format-specifiers** ( to specify the datatype ) within the double quotes [" "], and variableName outside the double quotes followed by a comma[ , ].

printf(" %formatSpecifier ", variableName);

#include< stdio.h >
void main()
{

int var1;
float var2;
char var3;
var1 = 56;
var2 = 23.46;
var3 = 'A';
printf("integer value = %d \nfloat value = %f \ncharacter value = %c",var1,var2,var3);

}

output

integer value = 56
float value = 23.46
character value = A

Variables

In any C-program we typically do lots of calculations. The results of these calculations are stored in computer’s memory(Like human memory). The computer’s memory also consists of millions of cells. The calculated values are stored in these memory cells. To make the retrieval and usage of these values easy, these memory cells(also called memory locations) are given names.

Since the value stored in each location may change, the names given to these locations are called variable names. In simple words, Variables are containers for storing data values like number and characters and the name given to that variable are called variableName.

rules for creating variableNames
  • A variable name is any combination of 1 to 26 alphabets, digits or underscores.
  • The first character in the variable name must be an alphabet or underscore ( _ ).
  • No commas or blanks are allowed within a variable name.
  • No special symbol other than an underscore can be used in a variableName.
  • Names can contain letters, digits and underscore.
  • Names are case sensitive.
  • Reserved keywords can not be used as names.

scanf();

In C-programming scanf() is used to take input from the user. In double quotes the format-specifier is declared then outside the double quotes the variableName followed by comma[ , ] and the ampersand symbol[ & ].

ampersand[ & ] is a symbol used in scanf to determine the address of the variable in the computer's memory.

scanf(" %formatSpecifier ", &variableName);

#include< stdio.h >
void main()
{

int var1;
float var2;
char var3;
scanf(" %d ",&var1);
scanf(" %f ",&var2);
scanf(" %c ",&var3);
printf("integer value = %d \nfloat value = %f \ncharacter value = %c",var1,var2,var3);

}

output

56
23.46
A

integer value = 56
float value = 23.46
character value = A



Format specifiers

format specifiers are special characters followed by % symbol to determine the datatype of the variable in the printf() and scanf() functions.

specifier datatype

%f float

%u unidentified



ASCII code

ASCII stands for American Standard Code for Information Interchange. It is a character encoding standard that uses numeric codes to represent characters. In ASCII, each character is assigned a unique number between 0 and 127. Common characters, such as letters, digits, and punctuation marks, have specific ASCII codes.
